Risks

There will be many risks in our Earth-Matrix software, our team works together to identify some specific risks and try to prevent them.



Some of our geopolitical reports that our Earth-Matrix generates or our geopolitical expert’s design may break laws in some countries. For example, it is illegal to make any comments to the communist party in China and make comments to North Korea’s region either. Besides, some reports may touch on political correctness and conflicts among cultures or religions, the negative influence may occur after our reports have been released.



Our software may face data breach from our database. Some government’s intelligence institutions, cyber armies, and some professional hackers may operate cyber attacks toward our database and cloud to steal data that we provide from our users, especially enterprise users.


We may face some lawsuits on intellectual property because data science technology may collect data from some companies or the institution’s databases. Some data may not be suitable for civil use even if our Earth-Matrix software may collect them automatically, so we may also face restrictions from some governments.


Supervised learning algorithms like regression, decision tree, random forest, and others; Machine learning algorithms like linear regression, logistics regression, Naive Bayes, K-Means, Random Forest, dimensionality reduction algorithms, gradient boosting algorithms; Deep learning like deep neural networks, deep belief networks, convolutional networks and so on. Skills and Jobs


Our Earth-Matrix software will go into an IPO in the future to bring profit to our shareholders from venture capitalists’ funds. Our A-stage of Earth-Matrix software needs four positions to develop, and here are some descriptions of these four positions.





The first position is the data scientist. This job needs to capture geopolitical data in the target nation or region, four stages like political data acquisition, data entry, signal reception, data extraction. Then, this job needs to process those geopolitical data like data mining, clustering, data modeling, data summarization. Next, communication like data reporting, data visualization, geopolitical information intelligence, decision-making methodology is significant. Moreover, data analysis like exploratory, predictive analysis, regression, text mining, qualitative analysis is necessary. Finally, data maintenance like data warehousing, data cleaning, data staging is what this job requires as well.


Some basic skills for data scientists in our work, programming language like R, Python, Apache Hadoop, MapReduce, SQL, databases like Apache Spark, NoSQL databases, cloud computing, D3, and other tools like data visualization software Tableau.




The second position is a UX designer. This job requires active participation in brainstorming and ideation sessions in the user interface, rapid conversion output from brainstorming and ideation sessions into workable wireframes, concept designs, and prototypes. Then, it requires strong knowledge of current design and prototyping tools for web, mobile, and media.


Some software like Adobe Photoshop, InDesign, Adobe Premiere Pro, Illustrator, Adobe XD, Unity, Unreal 3, Unreal 4,Davinci, Balsamiq and other softwares are needed to be mastered.




The third position is algorithm engineering, our Earth-Matrix software needs to use some algorithms to let the Earth-Matrix run. Not only in data collection, but also in inner messaging, inner logical framework, functions like public opinion monitor systems, war predictions analysis and others all need professional algorithms.




Some programming languages like C++, Java, Python, C#Swift, R language all need to be mastered in our Earth-Matrix. Some developing tools like Netbeans, R studio, Microsoft Visual Studio, Apple Xcode, Microsoft Azure, Data Studio should be used in this job position.


The fourth position is geopolitical experts. This position requires abundant historical knowledge and political knowledge.
